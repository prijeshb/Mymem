# V1-0001 — Feature Summary

Branch: `V1-0001`
Base: `master`
Status: Complete

---

## Overview

Foundation sprint — RAG infrastructure, router refactor, ingest pipeline improvements,
frontend UX polish, and developer workflow fixes.

---

## Backend

### RAG System (`mymem/rag/`)
- `store.py` — sqlite-vec chunk store with vector similarity search
- `pdf_parser.py` — pypdf extraction with paragraph-aware sliding-window chunking
- `embedder.py` — Ollama `nomic-embed-text` embeddings (768-dim)
- `ingest.py` — orchestrates parse → embed → store pipeline
- Local PDFs now skip LLM idea extraction entirely — RAG-indexed only for search-time retrieval

### Router Refactor (`mymem/pipeline/router/`)
- Split monolithic `router.py` into a package: `_router`, `_chain`, `_cost`, `_registry`, `_types`, `_utils`
- Model config updated to `gemma4:31b-cloud`

### Ingest Pipeline (`mymem/pipeline/ingest.py`)
- Uploaded files saved to `raw/<source_type>/` permanently (was: tempfile, deleted after ingest)
- Local PDFs short-circuit after security scan → RAG-index only, return `rag_only=True`
- `IngestResult` gets `rag_only` + `rag_chunks` fields
- YouTube ingest analytics wired in

### Ingest Analytics (`mymem/observability/ingest_analytics.py`)
- Quality tracking for YouTube ingests

### Query Pipeline (`mymem/pipeline/query.py`)
- Hybrid retrieval: wiki keyword search + RAG vector search combined

### Search (`mymem/pipeline/search.py`)
- DDG + Wikipedia fallback + TF-IDF Phase 2

### Web Routes (`mymem/web/routes/`)
- `logs.py` — new dedicated module for `GET /api/log` and `GET /api/heatmap` (extracted from `api.py`)
- All upload files now persist to `raw/<subdir>/` with conflict-safe naming
- `api.py` — expanded with archive/restore/RAG/analytics/daily endpoints

### Dev Mode (`mymem/web/app.py`, `mymem/cli.py`)
- `mymem serve --dev` — skips `frontend/dist/` serving, adds CORS for Vite dev server, enables uvicorn reload
- Root `/` in dev mode returns JSON with UI URL instead of 404

## Frontend

### New Components
- `ClaudeLoader` — three bouncing dots (Claude-style) with ingest-specific witty words; used only in `IngestPage`
- `CitationChip` — inline citation display

### Updated Components
- `LoadingSpinner` — replaced HeroUI `Spinner` with custom CSS spinning ring; witty cycling words for all non-ingest loaders

### Dashboard (`DashboardPage.tsx`)
- Removed session cost from Overview card
- Removed Recent Activity card (log data moved to dedicated logs route)

### Ingest Page (`IngestPage.tsx`)
- `ResultCard` shows distinct UI for RAG-only PDF indexing (sky-blue "Indexed for search" card)
- File upload persists to `raw/` — no more temp files

### Dev Workflow (`vite.config.ts`)
- Port fixed to `5174`
- SSE proxy configured with `x-accel-buffering: no` for streaming Q&A

### Wiki Page (`WikiPage.tsx`)
