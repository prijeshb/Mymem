# MyMem — Specs Index

All product and feature specifications live here.
Every new feature **must** have a spec before any code is written.

---

## How to Use Specs

1. **New feature request** → create `docs/specs/feat_<name>.md` from the template
2. **Review spec** with user before writing any code
3. **Reference spec** during implementation — it is the source of truth
4. **Mark spec status** when done: `Status: Shipped` + date

---

## Spec Template

Copy `docs/specs/_template.md` for every new feature spec.
